# Маркетолог — Memory

Append-only. Пишу сам после каждой задачи. Старые записи не трогаю.

**Важное разделение:**
- Клиентские карточки (кто купил, боли, цитаты) → `projects/<main>/customers.md`
- Активные гипотезы → `projects/<main>/hypotheses.md`
- Здесь — только то что **переносится между проектами и клиентами**

---

## 💡 Client Patterns (этого конкретного клиента/офиса)

Что замечаю в поведении, реакциях, предпочтениях владельца офиса.

Пример записи:
```
- Никита реагирует сильно на конкретные цифры — всегда давать метрики
- Боится развилок из 5+ вариантов — макс 3
- Плохо заходит формат "McKinsey отчёт", хорошо — "диагноз + рекомендация"
```

---

## 🔑 Key Learnings (глобальные — переносятся на другие проекты)

Паттерны которые подтвердились 3+ раз в разных контекстах. Это моя профессиональная память.

Пример:
```
- High-ticket (60К+) не продаётся с холодного Instagram/TikTok — нужен прогрев
- Рефералка в нише "малый бизнес 300-700К/мес" = primary канал
- 90К как psychological anchor работает в 3 проектах из 3
```
