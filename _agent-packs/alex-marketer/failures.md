## Pre-loaded failures (общие уроки v2.0 — наследуются всем установкам)

### 2026-04-28 — PUNDIT вместо VoC (Phase B fundamental fail)

**Что предположил:**
vc.ru-посты с >100 лайков = жирная VoC. Источники с высокой социальной активностью автоматически считал релевантными. Source quality multiplier для vc.ru post = 3 (на уровне реальных комментов).

**Что оказалось на деле:**
В Phase B для НейроШтаба собрал 90 «цитат» с vc.ru. Из них:
- ~30 — аналитические посты экспертов *про* ЦА (Илья Ершов про AI, Шишов про предпринимателей, Кузьмина про маркетологов) — **это PUNDIT, не голос ЦА**
- ~15 — статистика и исследования (McKinsey, %% опросов) — **это STATISTIC, не VoC**
- ~10 — CEO больших компаний про общие проблемы (Ларьяновский Skyeng) — **это голос корпоративного CEO, не голос эксперта-в-потолке**
- ~30 — близкое к VoC, но всё ещё аналитический формат
- ~5 — настоящие VoC

Клиент глянул и сразу сказал: «это не мой голос ЦА» — был прав. Ниша эксперта (психологи / коучи / нутрициологи) на vc.ru НЕ пишет — она в комментах под видео конкурентов на YouTube, в Telegram-каналах ниш, в b17.ru / profi.ru / kursometr.ru.

**Корневая причина:**
1. Source quality multipliers были перекошены в сторону vc.ru/Habr (3-4 vs YouTube comment 1)
2. НЕ было фильтра «первое лицо + конкретика» на каждой цитате
3. Comments-mining под контентом конкурентов был размыт между субагентами 1, 3, 5 — никто его явно не делал как ПРИОРИТЕТНЫЙ

**Правило на будущее:**

1. **VoC = первое лицо + конкретика + реальный человек.** Перед попаданием в кластер каждая цитата проходит **3 теста** (см. `knowledge/voc-quality-rules.md` Шаг 3.5):
   - Тест 1: первое лицо («я», «у меня», «мой») — НЕ «73%», «эксперты», «многие»
   - Тест 2: конкретика (цифра, имя, событие) — НЕ обобщения
   - Тест 3: реальный человек, не пунди́т (атрибуция к клиенту, не к аналитику)

   3 из 3 → `gold_voc` | 2 из 3 → `signal_voc` | 1 или 0 → `pundit`/`statistic` (отдельные файлы, НЕ в основной портрет)

2. **Иерархия источников переписана** (см. `knowledge/voc-quality-rules.md` секция "Приоритет источников"):
   - 🥇 YouTube/TG/IG comments под контентом конкурентов = multiplier **5** (главный источник VoC)
   - 🥈 Нишевые форумы (b17, profi, kursometr) = multiplier 4
   - 🥉 vc.ru/Habr с автором-клиентом (не аналитиком) = multiplier 2
   - 🔴 vc.ru-аналитический пост = multiplier 1, помечать `pundit`
   - 🚫 Statistic / McKinsey / опросы = multiplier 0, в `market-stats.md`

3. **Subagent 5 (Конкурентная карта) теперь ОБЯЗАН делать comments-mining** под контентом топ-5 конкурентов на YouTube/Telegram/Instagram. Минимум 30 перволичных цитат комментов. Это главный источник VoC.

4. **Жёсткий gate в БПСВ-синтезаторе**: `gold_voc < 25` → STOP, эскалация. `dominant_source_type > 50%` → STOP, эскалация (диверсификация обязательна).

5. **3 файла на выходе** вместо 1:
   - `voice-of-segment.md` — только gold_voc + signal_voc (настоящая VoC)
   - `pundit-analysis.md` — аналитика экспертов про ЦА (контекст рынка, не цитаты)
   - `market-stats.md` — статистика и исследования (фон для лендинга)

**Связанные изменения:** `knowledge/voc-quality-rules.md` создан как закон. `audience-internet-research/SKILL.md` Section 1.0 + Subagent 2 + Subagent 5 + Шаг 4 Match-методики + Section 11 anti-patterns переписаны.
