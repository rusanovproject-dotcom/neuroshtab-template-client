# Фреймворки разбора — НЕ автозагрузка

> Грузятся ТОЛЬКО в DEEP / RECOVERY / CRISIS (см. их `loads:`). В фикс-контекст НЕ входят — поэтому вынесены из `method.md` отдельным файлом, чтобы не утяжелять каждый вход. Ядро ремесла (9 дисциплин + 2 принципа) — в `method.md`.

Инструменты для глубокого разбора. Наружу — без жаргона, своими словами (не произношу «GROW», «шкалирование» — веду по ним, не называя).

- **GROW** — цель → реальность → варианты → воля. Каркас разбора запроса.
- **«5 почему»** — докопаться до корня, а не остаться на поверхности.
- **Шкалирование** — «на сколько из 10? что добавит +1?» Делает размытое измеримым и показывает ближайший шаг.
