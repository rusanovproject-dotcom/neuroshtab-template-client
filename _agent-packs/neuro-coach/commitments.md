# Привычки

То, что человек держит регулярно. Главная привычка системы — «Ежедневный чек»: она предзасеяна и есть у каждого. Её стрик — центральное число, которое человек видит каждый день.

Формат строки:
`привычка | N/нед | стрик | лучший | мес% | зачем (→goal/drive) | ставка`

**Три правила стрика:**

1. Стрик считается за СДЕЛАННЫЙ ЧЕК (полный или минимальный), НЕ за выполнение задач. Закрыл чек — +1, даже если из трёх задач не сделал ни одной. Стрик — про связь с собой, задачи — про эффективность. Не смешивать.
2. Истина = детерминированный пересчёт из `tracking/metrics-YYYY.jsonl` по календарным датам (алгоритм freeze/rest/grace — `tracking/procedures.md`, секция «Пересчёт стрика»). Кеш `стрик`/`лучший` в таблице выше — только для показа; на каждом чеке сверяется с пересчётом, расходятся — кеш переписать.
3. Слом стрика → любопытство, не приговор. Не «ты провалился», а «что случилось». Вернулся — это уже навык, цифру не реанимируем, обрамляем заново. Милстоуны 7 / 30 / 100 — повод отметить.
