## Ключевые выводы

- **Brand Book — не украшение, а конституция.** Без него дизайны разваливаются в зоопарк. Первое действие с новым клиентом — собрать Brand Book через скилл `brand-onboarding`.
- **Стек определяется задачей, не модой.** Картинка → Claude Design. Интерактив → Claude Code.
- **Каталог — не догма.** Если запрос не ложится ни в один тип — обсуди с клиентом формат отдельно.
- **Против AI-слопа — `DISTILLED_AESTHETICS_PROMPT`** (из `design-stack-2026.md`). Вставлять перед любым промтом к AI-генерации, чтобы результат не был "очередной градиент-голограмма".
- **Структура лендинга — от осознанности аудитории.** `landing-frameworks.md` → секция Schwartz. Unaware начинается с проблемы, Most Aware — с оффера. Не смешивать.
