# Designer — install manifest

Машиночитаемые метаданные для установки пака через скилл `/install-agent designer`.
Скилл читает этот файл, копирует файлы по указанным путям, и вставляет указанные строки в базовые конфиги офиса.

## Metadata

```yaml
agent_id: designer
agent_name_human: Дизайнер
agent_name_in_chat: Дизайнер
short_role: Визуальный партнёр — собирает Brand Book, сам делает лендинги / дашборды / обложки / презентации через Claude Code артефакты + 4 опциональные MCP-библиотеки (Magic UI / ReactBits / Aceternity / 21st.dev Magic) на 400+ готовых компонентов и Claude Design. Деплоит на Vercel одной командой.
trigger_keywords: [дизайн, визуал, лендинг, презентация, обложка, брендбук, дашборд, инфографика, one-pager, КП, карточка, задеплой, выложи сайт]
version: 3.0.0
requires: []
optional_mcp_servers: [magic-ui, reactbits, aceternity-ui, 21st-magic]
```

**Что нового в 3.0.0:**
- Подключение 4 MCP-серверов (Magic UI, ReactBits, Aceternity UI, 21st.dev Magic) на 400+ готовых компонентов через `.mcp.json` в корне офиса
- 5 новых knowledge-файлов: `mcp-libraries.md`, `component-decision-matrix.md`, `animation-patterns.md`, `landing-playbook.md`, `mcp-setup.md` (готовая инструкция-шаблон для пользователя как подключить MCP за 5 минут)
- В `core.md` — поведенческие принципы: think-aloud (с антипримерами), варианты вместо одного пути (с твоей рекомендацией), проактивная самодокрутка (только на ключевых секциях), антипаттерн «шаблонная сборка» (с критериями отказа от MCP), Pro Upgrade Pack (вшит в main flow перед деплоем)
- В первом контакте — проверка подключённости MCP и предложение пользователю подключить если нет
- Чистка устаревших guardrails из v2.x (противоречие «не запускай генерацию сам» vs «делаешь всё сам» убрано)

## MCP Libraries setup (опционально, но рекомендуется)

Дизайнер работает «из коробки» — собирает лендинги через Claude Code артефакт на голом Tailwind + shadcn. Это базовый путь, его достаточно для простых задач.

**Но есть множитель силы — 4 MCP-сервера на 400+ готовых компонентов.** С ними лендинги собираются в 3-5 раз быстрее и выглядят жирнее (Aurora WebGL-фоны, spotlight-эффекты, shimmer-кнопки, bento-grid карточки, marquee-логотипы, генерация UI из текста).

**Как подключить (5 минут):**

1. В корне офиса должен лежать файл `.mcp.json` со следующим содержимым:

```json
{
  "mcpServers": {
    "magic-ui": {
      "command": "npx",
      "args": ["-y", "@magicuidesign/mcp@latest"]
    },
    "reactbits": {
      "command": "npx",
      "args": ["-y", "reactbits-dev-mcp-server"]
    },
    "aceternity-ui": {
      "command": "npx",
      "args": ["-y", "aceternityui-mcp"]
    },
    "21st-magic": {
      "command": "npx",
      "args": ["-y", "@21st-dev/magic@latest"],
      "env": {
        "API_KEY": "YOUR_21ST_DEV_API_KEY_HERE"
      }
    }
  }
}
```

2. Если `.mcp.json` уже есть — добавь четыре сервера в существующий блок `mcpServers`. Не удаляй то что там было.

3. Для **21st.dev Magic** — получи API-ключ на https://21st.dev/magic и вставь вместо `YOUR_21ST_DEV_API_KEY_HERE`. Остальные три сервера работают без ключей. Если ключа нет — оставь placeholder, 21st-сервер просто не запустится, остальные три будут работать.

4. Перезапусти Claude Code (закрой и открой). При первом запуске npx скачает серверы — это разовая операция на 30-60 секунд каждый.

5. Проверка: спроси Дизайнера «какие у тебя сейчас MCP-серверы для дизайна?». Он посмотрит свои инструменты и ответит конкретно.

**Безопасность:**
- `.mcp.json` уже добавлен в `.gitignore` (содержит твой API-ключ — не коммить).
- В репо лежит только `.mcp.json.example` без ключа — это шаблон для других клиентов.
- Если API-ключа нет — Дизайнер всё равно работает, просто без 21st.dev Magic генерации.

**После подключения — Дизайнер сам:**
- В первом контакте упомянет что MCP подключены, покажет какие именно
- Будет использовать `mcp__magic-ui__*` / `mcp__reactbits__*` / `mcp__aceternity-ui__*` / `mcp__21st-magic__*` через прямой вызов
- Адаптирует каждый компонент под твой Brand Book (палитра, шрифты, радиусы)
- Думает вслух: «вижу 3 варианта для hero — Aurora WebGL, spotlight, animated grid. Под твой стиль ближе Aurora. Берём?»
