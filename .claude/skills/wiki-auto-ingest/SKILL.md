# wiki-auto-ingest — тихая пакетная обработка очереди

> Отличие от `/wiki-ingest`: тот спрашивает подтверждение перед записью. Этот — **не спрашивает**,
> но страхуется confidence-флагами и обязательным lint. Источник правил тот же: `wiki-ingest/SKILL.md`.

## Когда запускается
- Вручную: `/wiki-auto-ingest` (обработать всё из `knowledge/_ingest-queue.md`).
- Из `/loop`: периодически разгребать очередь.
- Авто-детект: хук `.claude/hooks/knowledge-autoingest.sh` кладёт новое сырьё в очередь при старте сессии — дальше запусти `/wiki-auto-ingest`.

## Workflow
1. Прочитай `knowledge/_ingest-queue.md` → секция `## Pending`. Если пусто — скажи «очередь пуста», стоп.
2. Для каждого pending-файла (бери пачкой до 5 за раз, чтобы не жечь контекст):
   - прочитай источник, определи домен (`knowledge/domains/<x>/`; AI → `domains/ai`);
   - нарежь на атомарные заметки по формату `wiki-ingest/SKILL.md`;
   - **в каждой заметке проставь `confidence: high|medium|low`** во frontmatter (насколько уверенно факт извлечён);
   - заметки с `confidence: low` продублируй путём в `domains/<x>/_review/` (mkdir -p) — чтобы человек глянул только их;
   - проставь `[[ссылки]]`, обнови `_MOC` домена;
   - НЕ трогай raw-источник.
3. Обнови `knowledge/INDEX.md` (точка входа домена) + допиши `knowledge/log.md` (`## дата — auto-ingest — <файл>`).
4. Отметь файл в очереди: `- [ ]` → `- [x] ... processed:<дата> notes:<N> confidence:<...>`.
5. **Обязательно прогони lint** (логика `/wiki-lint`): висячие ссылки → стабы/фикс, сироты → ссылки/MOC. Граф должен остаться чистым.
6. Отчёт: сколько файлов, сколько заметок, сколько в `_review/` (low-confidence на проверку человеком).

## Guardrails
- Не выдумывать. Нет в источнике → не пишем. Сомнение → `confidence: low` + в `_review/`.
- Raw неприкосновенен. Provenance в каждой заметке.
- Всегда lint после пачки — авто-режим без человека обязан оставлять граф чистым.
- Дорого? Бери Haiku/Sonnet для ingest, не Opus.
- Антирекурсия (для headless): процесс выставляет `KNOWLEDGE_AUTOINGEST=1`, детект-хук это видит и молчит.

## Output Contract
- **Отчёт о пачке** — N файлов → M заметок, K в `_review/`, граф чистый/проблемы.
- **Пусто** — «очередь пуста».
