# Skills _archive

Сюда переезжают **устаревшие скиллы** — те что заменены новыми, но удалять которые жалко (история, точечный reference).

**Правила:**
- Файлы здесь read-only логически. Изменять только если меняется deprecation note.
- Каждая папка имеет суффикс `-superseded` или `-deprecated` в имени.
- В `SKILL.md` каждого архивного скилла — frontmatter с `status: deprecated` и `superseded_by:` ссылкой на актуальный скилл.
- Архивные скиллы НЕ запускаются — Claude Code их не индексирует благодаря `_` в имени папки `_archive/` (служебный префикс).
