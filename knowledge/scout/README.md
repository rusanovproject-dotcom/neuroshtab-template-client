# Scout — досье по сферам

Эта папка хранит досье, которые собирает скилл `/scout <домен>`.

## Для чего

Когда надо быстро разобраться в новой для тебя сфере — запусти `/scout <домен>`. Помощник за 10-15 минут соберёт:
- **Топ-практиков** сферы — с блогами, подкастами, GitHub
- **Ключевые книги и методологии** — классику и свежие подходы
- **Публичные промпты и скиллы** — что уже сделали другие, что можно взять
- **MCP / инструменты** — чем пользуются профессионалы
- **Примеры** — как выглядит эталон в индустрии
- **Рекомендация топ-5** — что взять сразу

## Когда полезно

- Перед тем как заказать нового помощника в офис (`/install-agent`) — понять какие возможности есть
- Когда заходишь в новую для себя тему (например, email-маркетинг, видео, продажи)
- Перед запуском проекта в незнакомой сфере — чтобы стартовать с опорой на мировой топ, а не с колена
- Когда хочется понять «чем я вооружён» по сравнению с конкурентами

## Как запустить

```
/scout копирайтинг
/scout дизайнер
/scout email-маркетинг
/scout видеомонтаж
/scout продажи
```

Или просто в чате: «разведай домен X», «что в мире про Y», «соберём досье по Z».

## Что получишь

1. **Файл-досье** здесь, в `knowledge/scout/<домен>-YYYY-MM-DD.md` — полный отчёт
2. **Короткое резюме в чат** с топ-находками и рекомендацией топ-5

Досье актуально **30 дней**. После — запусти `/scout` заново, чтобы обновить.
