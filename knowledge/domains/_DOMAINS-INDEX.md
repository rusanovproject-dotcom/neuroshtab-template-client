# Domains — карта доменов знаний

> Атомарный слой вики (Karpathy LLM Wiki). Заметки создаёт `/wiki-ingest`, ревизует `/wiki-lint`.
> **Правило входа:** в любой домен заходим через его `themes/_MOC.md` — это карта темы со списком заметок.
> Не грепай дерево домена руками — читай `_MOC.md`, дальше иди по `[[ссылкам]]`.

## Домены

| Домен | Что внутри | Вход |
|-------|------------|------|
| **audience** | ЦА: аватары, боли, сегменты, JTBD | [audience/themes/_MOC.md](audience/themes/_MOC.md) |
| **product** | продукт: офферы, концепции, продуктовая лестница | [product/themes/_MOC.md](product/themes/_MOC.md) |
| **brand** | бренд: голос, визуал, позиционирование | [brand/themes/_MOC.md](brand/themes/_MOC.md) |
| **competition** | конкуренты, рынок, разведка | [competition/themes/_MOC.md](competition/themes/_MOC.md) |

## Структура каждого домена
```
<домен>/
  themes/_MOC.md   — карта темы (вход, список заметок со ссылками)
  concepts/        — атомарные концепты/методы
  tools/           — инструменты
  people/          — люди/эксперты
```

> Пусто на старте — наполняется по мере работы офиса. Источник правил — `knowledge/CLAUDE.md`.
