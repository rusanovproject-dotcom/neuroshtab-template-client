# Команда — 11 агентов

| Агент | Роль | Когда зовём |
|-------|------|-------------|
| **Director** | Оркестратор, Chief of Staff | Всегда начинаем отсюда |
| **Стратег** | Персональный партнёр программы | На онбординге и для еженедельных чек-инов |
| **Producer** | Маркетинг, стратегия, офферы | Когда про продажи и позиционирование |
| **Copywriter** | Контент, тексты, посты | Написать любой текст |
| **Tech Lead** | Код, деплой, инфра | Технические задачи |
| **Designer** | Визуал, UI, вёрстка | Любой визуал или фронт |
| **Intelligence** | Разведка, тренды, конкуренты | Исследовать рынок |
| **Hermes** | Продажи, КП, follow-up | Работа с лидами и сделками |
| **Демиург** | Архитектор офиса | Когда нужно перестроить команду или процессы |
| **Брахма** | Ревьюер, критик решений | Когда нужна вторая пара глаз |
| **ПАРА** | Организатор, разбор инбокса | Когда бардак в папках |

---

## Как с ними работать

**Обычная задача** — через Director:
> "Привет, нужно написать пост про запуск нового продукта"
Director сам позовёт Copywriter, потом Designer (если нужна обложка), потом Брахму (ревью).

**Онбординг или ре-стратегия** — через Стратега:
```
/strategist-intake    # первичная распаковка
/strategist-unpack    # детальный разбор документов
/strategist-discovery # адаптивный диалог по 4 блокам
/strategist-roadmap   # финал: стратегия + 6 недель + HTML-roadmap
/strategist-checkin   # еженедельно
```

**Прямое обращение к агенту:** `@director`, `@producer`, `@copywriter`, `@strategist` и т.д.

## Layered memory

У каждого агента — два файла:
- `core.md` — ядро, обновляется из template при `/update-agents` (пока ручное обновление)
- `overrides.md` — твои персональные правила, НЕ трогаются при обновлении

Пиши свои кастомизации в `overrides.md` — они имеют приоритет.
