## 2026-03-22: Конвейер качества (Demiurg → Brahma → Cherry)

**What:** Три агента = три уровня контроля. Demiurg создаёт, Brahma проверяет (8 измерений), Cherry полирует (7 осей). Self-grading bias устраняется внешним ревью.
**Why:** Когда Demiurg сам оценивал свою работу (Judge в review-refine loop), оценки были завышены. Внешний Brahma даёт объективную картину.

## 2026-03-22: validate.sh как детерминированный hook

**What:** Bash-скрипт проверяет broken links, INDEX.md, размеры CLAUDE.md, TODO-заглушки, дубли -- за 2 секунды. Ловит 80% проблем без LLM.
**Why:** Из курса Сараева: "Hooks для детерминированных вещей, CLAUDE.md для judgment". Валидация ссылок -- детерминированная задача, не нужен LLM.

## 2026-03-22: Hierarchical routing для 10+ агентов

**What:** Четвёртый паттерн в folder-patterns.md: Director видит только 7 Heads, каждый Head -- свои 3-4 специалиста. Двухуровневый роутинг.
