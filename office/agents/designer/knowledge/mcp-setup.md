# MCP Setup — что отдавать пользователю когда он сказал «давай подключим»

**Контекст:** ты упомянул что у тебя есть 4 опциональные библиотеки на 400+ готовых компонентов. Пользователь сказал «давай подключим». Отдай ему **готовую инструкцию ниже** живым языком — не сырой JSON.

---

## Что говорить пользователю (шаблон сообщения)

> *«Подключаем 4 библиотеки. Шаги простые — 5 минут.*
>
> ***Шаг 1.** Открой в этом офисе файл `.mcp.json` (он в корне папки, рядом с CLAUDE.md). Если его нет — создай. Внутрь вставь это:*
>
> *```json*
> *{*
> *  "mcpServers": {*
> *    "magic-ui": {*
> *      "command": "npx",*
> *      "args": ["-y", "@magicuidesign/mcp@latest"]*
> *    },*
> *    "reactbits": {*
> *      "command": "npx",*
> *      "args": ["-y", "reactbits-dev-mcp-server"]*
> *    },*
> *    "aceternity-ui": {*
> *      "command": "npx",*
> *      "args": ["-y", "aceternityui-mcp"]*
> *    },*
> *    "21st-magic": {*
> *      "command": "npx",*
> *      "args": ["-y", "@21st-dev/magic@latest"],*
> *      "env": {*
> *        "API_KEY": "YOUR_21ST_DEV_API_KEY_HERE"*
> *      }*
> *    }*
> *  }*
> *}*
> *```*
>
> ***Шаг 2.** Для 21st.dev нужен бесплатный API-ключ. Получить здесь — https://21st.dev/magic. Вставь его вместо `YOUR_21ST_DEV_API_KEY_HERE`. Остальные 3 библиотеки работают без ключей. Если ключ получать лень — оставь placeholder, тогда 21st.dev просто не запустится, остальные 3 будут работать.*
>
> ***Шаг 3.** Закрой Claude Code и открой заново. При первом запуске он скачает библиотеки — это 30-60 секунд каждая, разовая операция.*
>
> ***Шаг 4.** Напиши мне «какие у тебя сейчас MCP-серверы?». Я посмотрю и подтвержу что всё подключилось.*
>
> *Безопасность: `.mcp.json` уже добавлен в `.gitignore` (твой ключ не уйдёт в git). В репо лежит только `.mcp.json.example` — шаблон без ключа.»*

---

## Что делать если у пользователя `.mcp.json` уже есть

Если в корне офиса уже есть `.mcp.json` с другими серверами (например, Notion MCP, Gmail MCP) — **не перезаписывай**. Скажи пользователю:

> *«У тебя `.mcp.json` уже есть с другими серверами. Не буду затирать. Открой файл, найди блок `"mcpServers": { ... }` и **добавь** туда четыре наших сервера через запятую после последнего существующего. Структура такая:*
>
> *```json*
> *{*
> *  "mcpServers": {*
> *    "твой-существующий-сервер": { ... },*
> *    "magic-ui": { "command": "npx", "args": ["-y", "@magicuidesign/mcp@latest"] },*
> *    "reactbits": { "command": "npx", "args": ["-y", "reactbits-dev-mcp-server"] },*
> *    "aceternity-ui": { "command": "npx", "args": ["-y", "aceternityui-mcp"] },*
> *    "21st-magic": { "command": "npx", "args": ["-y", "@21st-dev/magic@latest"], "env": { "API_KEY": "..." } }*
> *  }*
> *}*
> *```*
>
> *Если запутаешься — пришли мне содержимое `.mcp.json`, я подскажу как добавить.»*

## Проверка после подключения

После того как пользователь перезапустил Claude Code — проверь свои инструменты. Должны появиться:

- `mcp__magic-ui__*` (searchRegistryItems, getRegistryItem)
- `mcp__reactbits__*` (search_components, get_component, list_categories)
- `mcp__aceternity-ui__*` (search_components, get_component_info, get_installation_info)
- `mcp__21st-magic__*` (21st_magic_component_builder, 21st_magic_component_inspiration, 21st_magic_component_refiner, logo_search) — только если API-ключ вставлен

**Сообщи пользователю что подключилось:**
> *«Готово. Подключилось: Magic UI (150+ компонентов), ReactBits (135+ WebGL фоны), Aceternity UI (200+ premium). 21st.dev Magic — [подключился / не подключился — нужен API-ключ]. Можем делать следующий лендинг в полную силу — Aurora-фоны, spotlight, shimmer-кнопки. Что собираем?»*

## Если что-то не подключилось

**Самая частая причина:** не перезапустился Claude Code. Скажи пользователю закрыть полностью (Cmd+Q на Mac) и открыть заново.

**Вторая причина:** нет npx (Node.js не установлен). Проверка — пусть пользователь напишет в терминале `npx --version`. Если не отвечает — нужно поставить Node.js (https://nodejs.org, LTS-версия).

**Третья причина:** имя сервера конфликтует с уже существующим. Переименуй один из них.

Если ничего из этого — спроси пользователя что показывает Claude Code в Settings → MCP, попроси скриншот.
