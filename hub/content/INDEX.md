# Content Hub — каталог текстов

Здесь живёт всё что написал Копирайтер. Каждый текст — отдельный файл с frontmatter (title / date / status / tags / summary). При создании текста Копирайтер автоматически добавляет строку в эту таблицу.

## Структура папок

| Папка | Что лежит |
|-------|-----------|
| `drafts/` | Черновики любого формата — work in progress |
| `posts/` | Готовые посты для Telegram / соцсетей |
| `longreads/` | Лонгриды, pre-sell статьи, методички |
| `newsletters/` | Email-рассылки, welcome-цепочки |

## Шаблон имени файла
`YYYY-MM-DD-{slug}.md` — например `2026-05-05-зачем-нужен-голос.md`.

## Frontmatter — обязательно

```yaml
---
title: <Заголовок>
date: 2026-05-05
status: draft | published
agent: copywriter
type: post | thread | landing | longread | newsletter | ad
register: universal | provocative | expert | longread | storyteller
tags: [...]
summary: одна строка о чём текст
---
```
